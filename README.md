# ⚡ Project 4 — The Living Prototype
### UI/UX Design Internship | DecodeLabs 

---

## 📌 Project Overview

This project is part of the DecodeLabs UI/UX Design Internship.

Building on Project 3's Visual Identity, this project brings the SoilToSoul app to life using Figma Prototyping — connecting screens with realistic transitions and micro-interactions, so stakeholders can experience the flow before development.

---

## 🎯 Objective

Breathe life into static designs using Interactions, Smart Animate, and Prototyping links — creating a clickable demo that looks and acts like a real app.

---

## 🛠️ Tool Used
Figma (Prototype Mode)

## 🌱 Interactions Implemented

- Splash Screen → Tap "Get Started" → Product List (Smart Animate)
- Product List → Tap any Product Card → Product Details (Smart Animate)
- Product Details → Tap Back Arrow → Product List (Smart Animate)
- Product Details → Tap "Add to Cart" → Product List (Success flow)

---

## 🎨 Design Consistency

All screens follow the SoilToSoul Visual Identity from Project 3 —

- Leaf Green #4A7C59 | Earthy Brown #5C3D2E | Harvest Gold #C9A84C
- Cream White #F5F0E8 background (including prototype preview background)
- Playfair Display + Montserrat typography

---

## 🌿 Key Concepts Practiced

- Prototype Mode navigation
- Smart Animate transitions
- Multiple cards linked to a single detail flow (Happy Path strategy)
- Hotspot hints enabled for easy testing
- Custom preview background matching brand identity
